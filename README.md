# SUPERHOT 모작

사용자의 움직임에 따라 시간이 진행되는 게임 메커니즘을 구현

## 주요 특징
- VR 환경에서의 **슬로우 모션 전투 구현**
- **물리 기반 상호작용** (무기 집기, 던지기 등)
- **시간 정지/재개 메커니즘** 재현
- 간단한 **AI 적 캐릭터** 구현

## 사용 기술
- Unity (URP)
- XR Interaction Toolkit
- Oculus Quest 2
